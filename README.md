# 🚀 Resume ATS Screener System

## 📌 Overview

This project is an AI-based Resume Screening System (ATS) that helps analyze resumes, extract key information, and generate a score based on skills, experience, and other factors.

---

## 🎯 Features

* 📄 Resume Upload (PDF)
* 🔍 Skill Extraction
* 📊 Score Calculation (out of 100)
* 📋 Candidate Dashboard
* 🤖 AI Chatbot (basic)
* 📑 Resume Analysis View

---

## 🛠️ Tech Stack

* Frontend: React
* Backend: Spring Boot
* Database: MongoDB
* Tools: Git, GitHub

---

## ⚙️ How It Works

1. Upload resume
2. Extract text using backend
3. Identify skills and details
4. Calculate score
5. Store in database
6. Display results on dashboard

---

## 📊 Scoring System

* Skills → 30
* Experience → 20
* Projects → 20
* Communication → 20

Total Score = 100
